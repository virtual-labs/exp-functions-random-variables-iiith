Moment Generating Functions

Moment generating functions (MGFs) are useful for several reasons, particularly for analyzing sums of random variables. Before discussing MGFs, let's define moments.

Definition: The n-th moment of a random variable XXX is defined as E[Xn]E[X^n]E[Xn]. The n-th central moment of XXX is defined as E[(X−EX)n]E[(X - EX)^n]E[(X−EX)n].

For example, the first moment is the expected value E[X]E[X]E[X]. The second central moment is the variance of XXX. Other moments provide additional useful information about random variables.

The moment generating function (MGF) of a random variable XXX is a function MX(s)M_X(s)MX​(s) defined as:

MX(s)=E[esX] M_X(s) = E\left[e^{sX}\right] MX​(s)=E[esX]

The MGF of XXX exists if there is a positive constant aaa such that MX(s)M_X(s)MX​(s) is finite for all s∈[−a,a]s \in [-a, a]s∈[−a,a].

MGFs are useful for two main reasons. First, the MGF of XXX provides all moments of XXX, hence its name. Second, the MGF (if it exists) uniquely determines the distribution. That is, if two random variables have the same MGF, they must have the same distribution. This method is particularly useful when working on sums of independent random variables.

Different Views of a Function of a Random Variable (FRV)

There are several different but essentially equivalent views of a function of a random variable (FRV). We will present two of them, highlighting their differences in emphasis.

Assume we have an underlying probability space P=(Ω,F,P)P = (\Omega, F, P)P=(Ω,F,P) and a random variable XXX defined on it. Recall that XXX is a rule that assigns a number X(ζ)X(\zeta)X(ζ) to every ζ∈Ω\zeta \in \Omegaζ∈Ω. Now, consider a measurable real function g(x)g(x)g(x) of the real variable xxx.

First View (Y: Ω → RY)

For every ζ∈Ω\zeta \in \Omegaζ∈Ω, we generate a number g(X(ζ))=Y(ζ)g(X(\zeta)) = Y(\zeta)g(X(ζ))=Y(ζ). The rule YYY, which generates the numbers {Y(ζ)}\{Y(\zeta)\}{Y(ζ)} for random outcomes {ζ∈Ω}\{\zeta \in \Omega\}{ζ∈Ω}, is an RV with domain Ω\OmegaΩ and range RY⊂RR_Y \subset \mathbb{R}RY​⊂R. For every Borel set of real numbers BYB_YBY​, the set {ζ:Y(ζ)∈BY}\{\zeta : Y(\zeta) \in B_Y\}{ζ:Y(ζ)∈BY​} is an event. Specifically, the event {ζ:Y(ζ)≤y}\{\zeta : Y(\zeta) \leq y\}{ζ:Y(ζ)≤y} is equal to the event {ζ:g(X(ζ))≤y}\{\zeta : g(X(\zeta)) \leq y\}{ζ:g(X(ζ))≤y}.
